# C-.NET-WinForms_Game
# Game Cờ Cá Ngựa (Ludo Game)
# Giới thiệu
Ứng dụng game xây dựng bằng Windows Forms sử dụng C# .NET
Mô phỏng luật chơi truyền thống, cho phép người chơi tung xúc xắc và di chuyển quân cờ trên bàn cờ
# Công nghệ sử dụng
C#
.NET Framework/ .NET
Windows Forms (Winforms)
# Chức năng chính
- Tung xúc xắc ngẫu nhiên
- Di chuyển quân cờ theo số bước
- Luân phiên lượt chơi giữa các người chơi
- Kiểm tra điều kiện thắng
- Giao diện trực quan (UI Winforms)
